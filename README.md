# Análise de Gastos com Python

Projeto simples de análise de dados usando Python.

O programa lê um arquivo CSV com gastos e calcula algumas informações.

## O que o programa faz

- calcula o total gasto
- calcula a média de gastos
- mostra o maior gasto

## Tecnologias

Python
CSV

## O que aprendi

- leitura de arquivos CSV
- manipulação de listas
- cálculo de estatísticas simples

## Melhorias futuras

- gerar gráfico de gastos
- separar gastos por categoria
